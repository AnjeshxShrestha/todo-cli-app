# 📝 To-Do CLI App

A simple command-line To-Do list application written in Python. Tasks are stored in a local `todos.json` file.

## 🚀 Features
- Add new tasks
- Mark tasks as done
- Delete tasks
- Persistent storage with JSON

## 🛠️ Requirements
- Python 3.x
